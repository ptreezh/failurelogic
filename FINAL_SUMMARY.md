# 认知陷阱平台 - 最终项目总结报告

## 项目概述
认知陷阱平台是一个基于《失败的逻辑》理论的教育互动游戏，旨在通过模拟各种认知偏差场景，帮助用户识别和克服思维局限。

## 核心功能
1. **指数增长误区场景** - 揭示人类对指数增长的线性思维误区
2. **复利思维陷阱** - 展示复利效应的反直觉特性
3. **历史决策重现** - 重现挑战者号等历史决策失败案例
4. **推理游戏场景** - 通过互动游戏挑战思维局限

## 技术架构
- **前端**: HTML/CSS/JavaScript + 交互式游戏引擎
- **后端**: Python FastAPI + 数据处理引擎
- **API端点**: RESTful API提供场景数据和计算服务
- **部署**: 支持本地服务器和云部署

1. 启动API服务器: `cd api-server && python start.py 8082`
