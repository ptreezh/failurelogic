# 认知陷阱平台专家技能

## 概述

`cognitive-trap-expert` 是一个专门为认知陷阱教育平台开发设计的专家级技能。该技能整合了认知偏差教育、指数增长理解、系统思维训练等领域的专业知识。

## 适用场景

这个专家技能适用于：

- 开发认知偏差教育场景
- 设计指数增长误区纠正课程
- 构建复利思维训练模块
- 创建历史决策重现案例
- 优化用户交互体验
- 实现技术架构改进

## 核心功能

### 专业领域
1. **认知偏差教育设计** - 设计有效的认知偏差纠正课程
2. **技术架构实现** - 实现前后端分离的系统架构
3. **系统性能优化** - 优化平台性能和用户体验

### 工作流程
1. **需求分析与场景设计** - 分析需求并设计认知陷阱场景
2. **技术实现与开发** - 指导技术实现过程
3. **测试与验证** - 设计全面的测试方案
4. **部署与优化** - 提供部署和优化建议

## 使用方法

在处理认知陷阱平台相关任务时，可以调用此技能来获得专业的指导和最佳实践建议。

## 专业知识

技能内置了认知偏差分类、教学设计原则和技术最佳实践等专业知识库，确保开发出高质量的教育产品。
