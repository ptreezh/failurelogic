## 2. 增强认知偏差检测准确性

### 偏差检测矩阵
| 偏差类型 | 检测指标 | 触发条件 | 严重程度 |
|---------|----------|----------|----------|
| 线性思维 | 预期 vs 实际差异 >50% | 连续3次高估 | 高 |
| 确认偏误 | 选择性采纳信息 >70% | 忽略反证 | 中 |
| 时间延迟盲区 | 忽略延迟效应 | 短期导向决策 | 中 |
| 指数增长误解 | 指数vs线性预期 >100% | 复杂系统误判 | 高 |

### 检测算法优化
```
偏差置信度 = (触发次数 × 权重 + 模式匹配度) / 标准化因子
- 高置信度 (>0.8)：明确指出偏差
- 中置信度 (0.5-0.8)：提示可能性
- 低置信度 (<0.5)：积累模式数据
```

### 实施策略
1. **机器学习增强**：训练模型识别微妙模式
2. **多维度检测**：结合行为、时间和结果数据
3. **自适应阈值**：根据用户水平调整敏感度
4. **假阳性控制**：减少误报率
