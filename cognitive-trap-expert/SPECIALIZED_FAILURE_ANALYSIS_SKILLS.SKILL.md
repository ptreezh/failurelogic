# Specialized Failure Analysis Skills for Cognitive Trap Platform

## Skill 1: Linear Thinking Trap Analysis

### Analysis Framework
- **Identification Criteria**: Detect situations where users assume linear relationships in non-linear systems
- **Impact Assessment**: Measure the magnitude of linear thinking errors
- **Correction Strategies**: Recommend approaches to overcome linear thinking bias

### Analysis Components
```
Linear Thinking Error = |Expected Linear Outcome - Actual Non-linear Outcome| / Expected Linear Outcome
```

### Diagnostic Indicators
- Large discrepancies between expected and actual outcomes
- Assumptions of proportional cause-and-effect relationships
- Ignoring threshold effects and tipping points
- Extrapolation of trends beyond reasonable bounds

## Skill 2: Confirmation Bias Detection and Analysis

### Analysis Framework
- **Information Filtering Patterns**: Identify selective information gathering
- **Evidence Interpretation Bias**: Detect biased interpretation of ambiguous evidence
- **Hypothesis Persistence**: Measure resistance to changing beliefs despite contradictory evidence

### Analysis Components
```
Confirmation Index = (Confirmatory Evidence Weighted / Total Evidence Considered) × 100
```

### Diagnostic Indicators
- Seeking information that confirms initial hypotheses
- Dismissing contradictory evidence
- Overvaluing confirmatory information
- Premature commitment to initial positions

---

## Skill 3: Time Delay Blindness Assessment

### Analysis Framework
- **Short-term Focus Measurement**: Assess preference for immediate results
- **Long-term Impact Neglect**: Evaluate failure to consider delayed consequences
- **Temporal Discounting Analysis**: Measure how outcomes are discounted over time

### Analysis Components
```
Time Delay Sensitivity = (Immediate Benefit Weight / Delayed Benefit Weight) - 1
```

### Diagnostic Indicators
- Preference for immediate rewards over larger future benefits
- Underestimating time required for effects to manifest
- Failure to plan for delayed consequences
- Misjudging causal relationships due to time gaps

## Skill 4: Exponential Growth Misconception Analysis

### Analysis Framework
- **Growth Rate Underestimation**: Measure underestimation of compound effects
- **Doubling Time Miscalculation**: Assess understanding of doubling periods
- **Saturation Point Ignorance**: Identify failure to recognize carrying capacity

### Analysis Components
```
Exponential Underestimation = |Actual Exponential Value - Estimated Value| / Actual Exponential Value
```

### Diagnostic Indicators
- Consistent underestimation of compound growth
- Difficulty grasping doubling effects
- Misunderstanding of growth limits
- Linear thinking applied to exponential processes

---

## Skill 5: Complex System Simplification Error

### Analysis Framework
- **Reductionist Thinking**: Identify oversimplification of complex systems
- **Variable Isolation**: Detect ignoring interdependencies
- **Unidirectional Causation**: Assess failure to recognize feedback loops

### Analysis Components
```
Simplification Score = Number of Relevant Factors Ignored / Total Relevant Factors × 100
```

### Diagnostic Indicators
- Treating complex systems as simple cause-effect chains
- Ignoring feedback loops and circular causality
- Failing to recognize unintended consequences
- Overlooking interaction effects between variables

---

## Skill 6: Risk Assessment Distortion Analysis

### Analysis Framework
- **Probability Miscalibration**: Measure distortion in probability estimates
- **Outcome Magnitude Errors**: Assess miscalculation of consequence severity
- **Frequency Distortion**: Evaluate misperception of event likelihood

### Analysis Components
```
Risk Distortion Index = |Subjective Risk Rating - Objective Risk Level| / Objective Risk Level
```

### Diagnostic Indicators
- Overestimating low-probability risks
- Underestimating high-probability risks
- Emotional amplification of risk perception
- Availability heuristic affecting risk assessment

---

## Skill 7: Anchoring and Adjustment Failure

### Analysis Framework
- **Initial Anchor Influence**: Measure weight given to initial information
- **Insufficient Adjustment**: Assess degree of adjustment from anchor
- **Reference Point Dependency**: Evaluate susceptibility to irrelevant anchors

### Analysis Components
```
Anchoring Strength = |Final Estimate - Rational Estimate| / |Anchor Value - Rational Estimate|
```

### Diagnostic Indicators
- Insufficient adjustment from initial values
- Influence of irrelevant numerical anchors
- Resistance to revising initial impressions
- Over-reliance on initial information

---

## Skill 8: Sunk Cost Fallacy Recognition

### Analysis Framework
- **Past Investment Consideration**: Measure inclusion of sunk costs in decisions
- **Opportunity Cost Neglect**: Assess failure to consider alternatives
- **Investment Justification Pressure**: Evaluate desire to justify past investments

### Analysis Components
```
Sunk Cost Bias = (Decision Influenced by Past Costs / Total Decision Factors) × 100
```

### Diagnostic Indicators
- Continuing failing projects due to past investment
- Reluctance to abandon unsuccessful courses of action
- Justifying further investment based on past commitments
- Failure to evaluate options independently of history

---

## Skill 9: Hindsight Bias Measurement

### Analysis Framework
- **Predictability Overestimation**: Assess belief that outcomes were predictable
- **Outcome Memory Distortion**: Evaluate distorted recall of initial predictions
- **Causal Story Construction**: Measure creation of compelling post-hoc explanations

### Analysis Components
```
Hindsight Index = |Remembered Pre-event Probability - Actual Pre-event Probability| / (1 - Actual Pre-event Probability)
```

### Diagnostic Indicators
- Belief that outcomes were obvious in retrospect
- Distorted memory of initial expectations
- Compelling explanations that ignore uncertainty
- Confidence in predictability after knowing outcomes

---

## Skill 10: Groupthink and Social Influence Analysis

### Analysis Framework
- **Dissent Suppression**: Measure pressure to conform to group opinion
- **Illusion of Unanimity**: Assess false consensus perceptions
- **Mindguard Behavior**: Identify protection of group from adverse information

### Analysis Components
```
Groupthink Score = (Conformity Decisions / Total Decisions) × (Pressure to Conform Rating / Maximum Possible Pressure)
```

### Diagnostic Indicators
- Suppression of dissenting opinions
- Pressure to reach consensus quickly
- Dismissal of outside opinions
- Shared rationalization of decisions

---

## Skill 11: Overconfidence Assessment

### Analysis Framework
- **Calibration Testing**: Measure alignment between confidence and accuracy
- **Precision-Confidence Correlation**: Assess relationship between confidence and actual performance
- **Competence Illusion**: Evaluate overestimation of abilities

### Analysis Components
```
Overconfidence Score = Average Confidence Rating - Actual Accuracy Percentage
```

### Diagnostic Indicators
- Confidence exceeding actual performance
- Narrow confidence intervals that exclude correct answers
- Underestimation of task difficulty
- Superiority illusion compared to others

---

## Skill 12: Availability Heuristic Distortion

### Analysis Framework
- **Recency Effect**: Measure influence of recent events on probability estimates
- **Salience Impact**: Assess influence of vivid or memorable events
- **Search Set Bias**: Evaluate limited search for relevant examples

### Analysis Components
``` 
Availability Distortion = |Estimated Frequency - Actual Frequency| / Actual Frequency
```

### Diagnostic Indicators
- Recent events weighted too heavily
- Vivid examples overvalued in judgment
- Limited search for disconfirming evidence
- Media coverage influencing probability estimates
