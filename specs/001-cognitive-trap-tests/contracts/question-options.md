# 具体化题目选项定义

## 2^200粒米存储空间题目

### 选择题选项
- A. 1万个足球场
- B. 100万个足球场  
- C. 1亿个足球场
- D. 以上都不对，不需要这么大
- E. 以上都不对，这些都不够

### 题目背景
如果2^200（约1.6×10^60）粒米需要存储，需要多大的仓库空间？

### 正确答案
E. 以上都不对，这些都不够

### 解释
2^200粒米的数量是1,606,938,044,258,990,275,541,962,092,341,162,602,522,202,993,782,792,835,301,376粒。这个数量远超宇宙中的原子总数（约10^80），因此即使是1亿个足球场也完全不够存储。

## 兔子繁殖题目

### 选择题选项
- A. 100年
- B. 50年
- C. 30年
- D. 20年
- E. 10年

### 题目背景
在一个类似地球的外太空星球上，宇航员带了5对（10只）兔子。兔子一年生长期，一对兔子一年可以繁殖10只，考虑兔子寿命，兔子如果没有外力影响增长速度约5倍。大约需要多少年达到与人类人口总数相当的80亿只兔子？

### 正确答案
E. 10年（实际计算为11年，但10年是最接近的选项）

### 解释
计算过程：10 -> 50 -> 250 -> 1,250 -> 6,250 -> 31,250 -> 156,250 -> 781,250 -> 3,906,250 -> 19,531,250 -> 97,656,250 -> 488,281,250 -> 2,441,406,250 -> 12,207,031,250，在第11年超过48亿只，第12年超过244亿只。指数增长的威力远超直觉，实际上只需要11年就能达到接近80亿只的规模。题目中的选项中最接近实际答案的是10年，这突显了人们对于指数增长的低估 - 大多数人会认为需要更长时间（如20年、30年甚至更多）。指数增长的威力远超直觉！
