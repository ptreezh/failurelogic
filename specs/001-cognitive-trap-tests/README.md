# 认知陷阱测试扩展

此特性扩展了认知陷阱平台，增加了指数增长误区、复利思维陷阱、历史决策失败案例重现和互动式思维陷阱游戏等功能。通过选择题和互动游戏的形式，让用户体验和学习各种认知偏差，特别是线性思维在面对指数增长和复利效应时的局限性。

## 特性概述

1. **指数增长误区测试**
   - 2^200规模问题
   - 米粒问题
   - 兔子繁殖问题

2. **复利思维陷阱测试**
   - 银行贷款利息比较
   - 复利vs线性增长对比

3. **历史经典决策失败案例**
   - 挑战者号发射决策
   - 泰坦尼克号航线选择

4. **互动式思维陷阱游戏**
   - 推理游戏
   - 思维局限暴露
