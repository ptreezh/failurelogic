# /speckit.specify 扩展规范：指数增长与认知偏差专项测试

**Feature Branch**: `failureLogic-specify-extension`  
**Created**: 2025-11-09  
**Status**: Draft  
**Input**: User request for enhanced testing of exponential growth misconceptions and linear thinking traps

## User Scenarios & Testing *(mandatory)*

### User Story 1 - 指数增长误区测试 (Priority: P1)

用户通过专项测试体验和学习指数增长误区，特别是对2^200这样的天文数字缺乏直观认识，以及对复合增长效应的低估。

**Why this priority**: 指数增长误区是现代决策中常见且严重的问题，影响从技术发展预测到金融投资决策的各个领域。

**Independent Test**: 用户完成关于2^200规模和兔子繁殖问题的测试，并收到关于指数增长误区的解释和反馈。

**Acceptance Scenarios**:

1. **Given** 用户开始指数增长误区测试, **When** 回答2^200规模的相关问题, **Then** 系统提供正确答案及解释
2. **Given** 用户面对兔子繁殖问题, **When** 估算达到100亿所需时间, **Then** 系统显示实际计算结果和指数增长的威力
3. **Given** 用户完成测试, **When** 查看结果反馈, **Then** 系统提供关于线性思维局限性的深入分析

### User Story 2 - 复利思维陷阱识别 (Priority: P2)

用户通过模拟场景识别和理解复利思维陷阱，学习如何在长期决策中考虑复合效应。

**Why this priority**: 复利思维在财务规划、职业发展和技能积累中至关重要。

**Independent Test**: 用户参与复利计算模拟，比较线性增长与指数增长的差异，并理解长期影响。

**Acceptance Scenarios**:

1. **Given** 用户进行复利计算, **When** 设置初始投资和收益率, **Then** 系统展示复利vs线性增长的差异
2. **Given** 用户查看长期影响, **When** 延长投资期, **Then** 系统突出显示复利效应的加速增长

### User Story 3 - 综合认知偏差评估 (Priority: P3)

用户通过综合性测试评估其在各种认知偏差方面的表现，特别是指数增长误区、时间延迟、确认偏误和系统性思考能力。

**Why this priority**: 综合评估能提供全面的认知能力画像，有助于个性化学习路径。

**Independent Test**: 系统提供多项选择题和情景题，评估用户在多个认知偏差维度的表现。

**Acceptance Scenarios**:

1. **Given** 用户开始综合评估, **When** 完成所有测试题目, **Then** 系统生成详细的认知偏差报告
2. **Given** 系统评估结果, **When** 生成个人化建议, **Then** 提供针对性的学习路径

## Requirements *(mandatory)*

### Functional Requirements

- **FR-001**: 系统 MUST 提供关于指数增长误区的专项测试，包含2^200规模的直观展示
- **FR-002**: 系统 MUST 包含兔子繁殖问题的计算，展示指数增长的惊人速度
- **FR-003**: 系统 MUST 提供复利计算器和可视化工具，展示线性vs指数增长差异
- **FR-004**: 系统 MUST 提供多项选择题测试，评估用户对指数增长的理解
- **FR-005**: 系统 MUST 提供情景模拟，让用户体验时间延迟和系统性思维失败
- **FR-006**: 系统 MUST 生成个人化认知偏差报告，指出用户的思维误区
- **FR-007**: 系统 MUST 提供针对性的学习材料，帮助用户克服特定认知偏差

### Key Entities

- **指数增长测试**: 包含测试ID、题目内容、正确答案、用户回答、评分等属性
- **认知偏差报告**: 包含用户ID、各项偏差评分、改进建议、学习路径等属性
- **复利计算器**: 包含本金、利率、时间、线性结果、复利结果等属性

## Success Criteria *(mandatory)*

### Measurable Outcomes

- **SC-001**: 用户能够理解2^200是一个天文数字（约1.6×10^60），比全宇宙原子总数还多
- **SC-002**: 用户能够正确估算指数增长问题，如兔子繁殖问题的实际时间
- **SC-003**: 用户能够识别复利与线性增长的差异，并理解长期影响
- **SC-004**: 用户完成测试后，对自身认知偏差有更深入的认识
- **SC-005**: 系统能够准确评估用户在指数增长误区方面的表现

## 专项测试内容

### 指数增长误区测试题

1. **2^200有多大规模？**
   - 选项A: 几百万
   - 选项B: 几十亿
   - 选项C: 约1.6×10^60（比全宇宙原子总数还多）
   - 选项D: 无法计算

2. **2只兔子每年翻5倍，何时能超过100亿？**
   - 选项A: 50年
   - 选项B: 30年
   - 选项C: 15年
   - 选项D: 8年

3. **10万元年复利8%，30年后约为？**
   - 选项A: 34万元（线性估算）
   - 选项B: 100万元
   - 选项C: 317万元
   - 选项D: 500万元

### 情景思考题

1. **城市规划挑战**：您需要规划未来20年城市发展，人口以指数增长，您会选用线性增长方案还是指数增长方案？请解释原因。

2. **投资决策**：面对确定获得10万元和10%概率获得1000万元的两个选项，从期望值角度分析，并思考直觉可能与数学结果不同的原因。

### 反思与应用

1. 在日常生活中，您是否曾经低估了某些趋势的长期影响？
2. 当面对需要长期规划的决策时，您是倾向于线性思维还是考虑指数效应？
3. 如何训练自己更好地理解复杂系统中的非线性关系？
